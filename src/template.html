template-web-app

This template repository is a starting point for web applications

It should have some basic styling; use the Kanit font, and have a "caret" image above


It includes the following set-up:

File and directory structure

Webpack support for css files, images and fonts

HTML webpack plugin

Webpack-dev-server to allow live loading during development (current issue with html updates not live loading)

Webpack configurations separated for dev and prod

ESlint and Prettier installed, with conflicts dealt with

Babel loader configured

Jest installed (may be some jest/babel config still required)

npm scripts for build, automated dev updates, jest testing and jest watch added
